  >> Website açıklarını bulmak (WAScan) <<
 
  
  - Bu araç ile birçok alanda tarama ve zafiyet bulma işlemi gerçekleştiriliyor .Taranan kısımlar ve bulunabilen zafiyet türleri aşağıda mevcut .

	WAScan (W)eb (A)pplication (S)cannier is a open source application secırity scanner . It is designed to find various vulnerabilities using "black-box" method ,
  that  means it wont study  the source code  of web applications but will work like a fuzzer,scannig  the web pages of the deployed web application ,
  extracting links and forms and attacking the scripts , sending payloadas and looking for error messages, ...etc. WAScan is built on python 2.7 and can run on 
  any platform which has a python environmnet .
